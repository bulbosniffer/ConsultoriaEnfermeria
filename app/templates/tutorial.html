📖 Justificación de la Aplicación

Esta aplicación fue desarrollada debido a la necesidad de concentrar mensualmente la información de las consultas de enfermería. Anteriormente, esta tarea se realizaba mediante hojas de Excel individuales por cada personal, pero en los equipos disponibles no se cuenta con Microsoft Office, sino con OpenOffice, el cual no ejecuta correctamente las macros necesarias para automatizar el trabajo. Esta plataforma soluciona ese problema centralizando la información de forma segura y accesible.

1. ⚖️ Ingreso a la plataforma

Accede desde: consultoriaenfermeria.onrender.com
Inicia sesión con tu usuario y contraseña.

2. 🔍 Navegación general

  • Inicio: Bienvenida al sistema.
  • Registrar consulta: Llenar hoja de consultoría.
  • Formatos: Ver/subir/eliminar documentos.
  • Cerrar sesión

3. 📅 Registrar hoja de consultoría

  1. Llenar los datos del paciente.
  2. La edad se calcula automáticamente desde la fecha de nacimiento.
  3. Se asigna el grupo etario según la edad.
  4. Si el paciente es también jefe de familia, el nombre se copia automáticamente.

5. 🚀 Funcionalidades automáticas

  • Cálculo automático de edad.
  • Selección automática del grupo etario.
  • Copiado inteligente si el jefe de familia es el mismo paciente.
  • Alertas visuales amigables.

7. 🧰 Preguntas frecuentes

  • ¿Puedo editar un registro? No, solo puedes eliminarlo y volver a ingresarlo.
  • ¿Funciona en celular? Sí, la app es responsiva y está optimizada para móviles.
